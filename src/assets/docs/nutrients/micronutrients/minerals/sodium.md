# Sodium
Sodium (N) is an essential mineral element. Its main function is to maintain extracellular fluid volume and plasma osmolality.

## Important relationships

## Daily requirements
- Men: 420mg/day;
- Women: 320mg/day;
- Pregnancy: 350mg/day;
- Lactation: 310mg/day.
